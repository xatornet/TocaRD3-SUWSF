# Toca Race Driver 3 - Ultrawide Patch (For Use With SUWSF)

This is a respository of ultrawide patch for use with [SUWSF](https://github.com/PhantomGamers/SUWSF). Inside you will find `SUWSF.ini` with the patch source and a complete zip that includes the correct folder structure along with both SUWSF and UltimateASILoader.

## Installation and Configuration

- Download the zip file.
- Extract the contents of the zip file in to the game folder.
- [optional]Check `SUWSF.ini` for configuration. (not needed at the moment)

For regular SUWSF configuration see the [`configuration`](https://github.com/PhantomGamers/SUWSF#configuration) section on the SUWSF repo.

## Troubleshooting

See the [`troubleshooting`](https://github.com/PhantomGamers/SUWSF#troubleshooting) section on the SUWSF repo.
